# Blinkit Sales Analysis 📊

## Project Overview

This project analyzes Blinkit's sales data to uncover insights about product performance, customer preferences, outlet characteristics, and sales trends. The analysis is performed using Python and data visualization libraries to help understand factors affecting sales.

## Objectives

* Analyze overall sales performance.
* Identify top-selling product categories.
* Study the impact of outlet size and location on sales.
* Explore customer ratings and product visibility.
* Generate meaningful visualizations for business decision-making.

## Dataset

The dataset contains information such as:

* Item Identifier
* Item Type
* Item Fat Content
* Outlet Identifier
* Outlet Size
* Outlet Location Type
* Outlet Establishment Year
* Item Visibility
* Item Weight
* Sales

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Business Insights & Conclusions

## Key Insights

* Identified the highest revenue-generating product categories.
* Compared sales across different outlet locations.
* Analyzed the relationship between item visibility and sales.
* Evaluated outlet performance based on size and establishment year.

## Visualizations Included

* Sales Distribution
* Outlet-wise Sales Analysis
* Product Category Performance
* Correlation Heatmap
* Item Fat Content Analysis
